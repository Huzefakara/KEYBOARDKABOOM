Keyboard Shortcuts

Shortcut Key Description
General Shortcuts
Command + C Copy
Command + X Cut
Command + V Paste
Command + Z Undo
Command + Shift + Z Redo
Command + A Select All
File and Finder Shortcuts
Command + N New window
Command + O Open
Command + S Save
Command + P Print
Command + W Close window
Command + Q Quit application
Navigation Shortcuts
Command + Tab Switch between applications
Command + ` Cycle through open windows of the current application
Command + Space Open Spotlight search
Browser Shortcuts
Command + T New tab
Command + W Close tab
Command + R Refresh
Command + L Highlight the URL bar
Command + Click Open a link in a new tab
Screen Capture Shortcuts
Command + Shift + 3 Capture entire screen
Command + Shift + 4 Capture selected area
Command + Shift + 4 then Space Capture a specific window
