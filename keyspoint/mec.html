MacOS Shortcut Keys

Space bar (while a file is selected in Finder)
:Preview the selected file without opening it.

Command + Space, then type a math expression
:Use Spotlight as a calculator by typing math expressions.

Control + Command + Space
:Opens the emoji picker for easy emoji input.

Command + Tab
:Switch between open applications.

Control + Scroll Up/Down
:Zoom in or out on the screen content.

F11 or Mission Control (usually mapped to F3 or Control + Up Arrow)
:Show or hide the desktop, revealing or minimizing open windows.

Option + Click on Apple Menu, then select "System Information"
:Quickly access detailed information about your Mac.

Command + Shift + 4, then drag to select the area
:Capture a specific portion of the screen.

Control + Command + Q
:Quickly lock your Mac's screen.
