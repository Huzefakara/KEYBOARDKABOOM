Windows Shortcut Keys

Win + Pause/Break
:Opens the System Properties window.

Win + . (period) or Win + ; (semicolon)
:Opens the emoji picker.

Win + Shift + S
:Opens the Snipping Tool to capture a specific area of the screen.

Ctrl + Shift + Esc
:Opens Task Manager directly.

Win + Ctrl + D
:Creates a new virtual desktop.

Win + Ctrl + F4
:Closes the current virtual desktop.

Win + , (comma)
:Peeks at the desktop (temporarily hides open windows).

Win + Alt + D
:Shows or hides the date and time on the desktop.

Ctrl + Win + Tab
:Activates Task View, allowing you to switch between open windows and virtual desktops.

Shift + Right-Click on a Taskbar item
:Opens a new instance of the program.

Ctrl+Shift+left mouse click
:Starts the selected app as an administrator.
