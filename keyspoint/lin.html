Linux Shortcut Keys

Ctrl + Alt + Backspace
Restart X Server: This combination can be used to restart the X server, effectively restarting the graphical interface.

Alt + SysRq + R E I S U B
Safe System Reboot: This sequence is used to safely reboot the system even if it's unresponsive. Each letter stands for a specific action:
R: Switch the keyboard mode
E: Send the Terminate signal to all processes except for init
I: Send the Kill signal to all processes except for init
S: Sync all mounted filesystems
U: Remount all mounted filesystems as read-only
B: Immediately reboot the system

Ctrl + Alt + F1 to F6
Switch Between Virtual Terminals: You can use these key combinations to access different terminal sessions.

Ctrl + Alt + Arrow Keys
On some systems, this can be used to switch between workspaces or virtual desktops.

Alt + F2
Opens a "Run Application" dialog where you can quickly launch a command or application.

Alt + F7
Used to switch back to the graphical user interface (GUI) from a virtual terminal.

Ctrl + Alt + L
Locks the screen and requires a password to unlock. Useful for securing your desktop when you're away.

Alt + PrtScn
Capture the currently focused window.

Ctrl + Alt + Tilde (~)
Switch between windows of the same application in the GNOME desktop environment.

Ctrl + Alt + Plus/Minus
Change screen resolution in some Linux distributions.

Ctrl + Alt + Shift + Arrow Keys
Move the current window to a different monitor in a multi-monitor setup.
